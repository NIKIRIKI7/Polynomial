Документация библиотеки работы с полиномами

Общее описание

Библиотека предоставляет инструменты для работы с математическими полиномами:

  • Класс Point для представления точек
  • Класс Polynomial для работы с полиномами
  • Класс NewtonPolynomial для интерполяции

Класс Point

Назначение

Представляет точку в двумерном пространстве (x, y).

Основные методы

Метод Описание
Point(double x, double y) Конструктор с координатами
getX(), getY() Получение координат

Пример использования

Point p1 = new Point(1.5, 2.3);
System.out.println(p1.getX()); // 1.5

Класс Polynomial

Возможности

  • Арифметические операции
  • Вычисление значений
  • Строковое представление

Пример создания

// 3x² + 2x + 1
Polynomial p = new Polynomial(1, 2, 3);
System.out.println(p); // "3.00x² + 2.00x + 1.00"

Класс NewtonPolynomial

Интерполяция

Строит полином по заданным точкам

List points = Arrays.asList(
    new Point(1, 2),
    new Point(3, 5)
);
NewtonPolynomial np = new NewtonPolynomial(points);
System.out.println(np.evaluate(2.0)); // Интерполяция в точке x=2

Важные предупреждения

  • Деление на ноль в Polynomial.divide()
  • Точки с одинаковыми X в интерполяции
